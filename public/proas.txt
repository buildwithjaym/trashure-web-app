==================================================
PROJECT PURPOSE
==================================================

Trashure is an AI-powered circular economy platform.

It does not only answer:

“Where should I throw this?”

It answers:

“What can this material become?”

Residents can scan discarded items and receive circular recovery recommendations:

- Reuse
- Sell
- Donate
- Recycle

The platform should also connect residents with:

- Nearby junkshops that accept the detected material
- Junkshops’ local price per kilogram
- Schools or community partners requesting the material
- Material requirements and preparation instructions
- Collection or recovery opportunities

The partner network is more important than the AI scanner.

The AI creates interest, but the platform becomes useful when it can answer:

- Which nearby junkshop accepts this material?
- How much does the junkshop pay per kilogram?
- What condition should the material be in?
- Which school currently needs the material?
- Where should the resident bring it?

==================================================
IMPORTANT BASE UI RULE
==================================================

The project’s Shadcn components use Base UI, not Radix UI.

Do not use:

asChild

Using asChild caused:

- React warning that asChild was passed to a DOM element
- Nested button elements
- Hydration mismatch
- Invalid HTML
- Dropdown trigger errors

Incorrect:

<Button asChild>
    <Link href="/somewhere">Open</Link>
</Button>

Incorrect:

<DropdownMenuTrigger asChild>
    <button>Profile</button>
</DropdownMenuTrigger>

Correct navigation approach:

const router = useRouter();

<Button
    type="button"
    onClick={() => router.push("/somewhere")}
>
    Open
</Button>

Correct dropdown trigger approach:

<DropdownMenuTrigger
    type="button"
    className="..."
>
    Profile content
</DropdownMenuTrigger>

DropdownMenuTrigger already renders the button.

Do not nest a button inside DropdownMenuTrigger.

==================================================
DROPDOWN MENU ISSUE THAT WAS FIXED
==================================================

A previous dropdown caused this error:

- button cannot be a descendant of button
- hydration failed
- React does not recognize asChild
- MenuGroupContext is missing

The causes were:

1. Using Radix-style asChild with Base UI
2. Nesting a button inside DropdownMenuTrigger
3. Using DropdownMenuLabel without the required menu group
4. Importing DropdownMenuSeparator more than once

The safe simplified dropdown should use:

- DropdownMenu
- DropdownMenuTrigger
- DropdownMenuContent
- DropdownMenuItem
- DropdownMenuSeparator

Avoid DropdownMenuLabel unless it is correctly wrapped by DropdownMenuGroup.

Keep imports unique. Do not import the same component twice.

==================================================
REACT HOOK RULE
==================================================

Hooks must remain inside the React component.

Incorrect:

const router = useRouter();

export default function RecyclerPage() {
}

Correct:

export default function RecyclerPage() {
    const router = useRouter();
}

Do not place useRouter, useState, useEffect, useMemo, or other hooks at module level.
