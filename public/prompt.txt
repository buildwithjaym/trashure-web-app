Project Context

You are helping build Trashure: Turn Trash Into Treasure.

Trashure is an AI-powered circular economy platform that prevents materials from becoming waste by identifying discarded items and recommending their highest-value recovery pathway.

The platform is not a simple recycling application.

Instead of asking:

"Where should I throw this?"

Trashure asks:

"What can this material become?"

The system helps users:

Identify materials
Understand their value
Choose circular actions:
Reuse
Sell
Donate
Recycle

The platform also provides LGUs with anonymous waste intelligence.

AI Integration Guide
Objective

When a user uploads an image:

The AI should answer:

What object is this?
What material is it made of?
Is it recyclable?
What can it become?
What action should the user take?
AI Workflow
Step 1

User uploads image.

Example:

plastic_bottle.jpg
Step 2

Save image to:

Supabase Storage

Bucket:

trashure-images

Example:

trashure-images/
 |
 |-- scans/
      |
      plastic001.jpg
Step 3

Send image to  Vision API.

Create:

app/api/analyze/route.ts

The API receives:

image_url

Then sends:

image
+
system prompt

to AI.

AI System Prompt

Use this:

You are Trashure AI,
a circular economy material intelligence assistant.

Analyze the uploaded waste image.

Your task:

1. Identify the object.
2. Identify the material type.
3. Estimate recyclability.
4. Suggest reuse possibilities.
5. Suggest selling opportunities.
6. Suggest donation opportunities.
7. Suggest recycling instructions.

Prioritize resource recovery.

Do not simply recommend disposal.

Return JSON format only.

Important Development Rules

When writing code:

Use TypeScript.
Use reusable components.
Use clean folder structure.
Do not hardcode AI results.
Store AI results in Supabase.
Validate uploaded images.
Handle AI errors gracefully.
Keep user privacy protected.
Store only anonymous waste analytics.
Design for future LGU adoption.

Final Goal

The final demo should show:

A user picks up a plastic bottle.

They scan it.

Trashure says:

"This is PET Plastic. It still has value."

Then shows:

Reuse.
Sell.
Donate.
Recycle.

The LGU dashboard updates:

"Plastic recovery increased by community participation."

This demonstrates a complete circular economy system, not just an AI scanner.


===============

The partner network is more important than the AI

The scanner creates initial interest, but the platform becomes useful only when it can answer:

Which nearby junkshop accepts this?
Which school currently needs this material?
Where is the nearest collection point?
How much is the local estimated value?
What condition must the material be in?
